# -
随着区块链技术的快速发展，其在扩展性与共识效率方面面临的瓶颈日益凸显。为了深入研究不同区块链架构对系统性能的影响，本文设计并实现了一个基于面向对象思想的区块链仿真系统。首先，系统采用分层架构设计，利用策略模式对共识模块进行解耦，实现了工作量证明（PoW）与权益证明（PoS）两种共识机制的动态切换与对比分析。其次，针对区块链扩容问题，系统在基础账本之上集成二层网络方案，设计了基于哈希时间锁定合约（HTLC）的双向支付通道模型，实现了链下资金的安全交互与即时结算。最后，本文构建了仿真测试环境，通过生成不同规模的随机交易流对系统进行了压力测试。实验结果表明：相较于 PoW 机制，PoS 共识在吞吐量（TPS）与资源消耗方面具有显著优势；同时，引入支付通道后，高频小额交易的确认延迟降低了两个数量级，且有效减少了主链的手续费成本。本系统结构清晰、扩展性强，为研究区块链底层原理及扩容技术提供了有效的实验验证平台。
